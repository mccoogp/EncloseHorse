This work is inspired by the game https://enclose.horse/. The file Play.py allows you to build a map and play it. MakeMap.py is a better version for just making a map. The majority of this project was about Compare.py and Showdifference.py which compare the performance of genetic algorithms for solving the optimal solution by comparing sexual and asexual forms of mutation in the algorithms. Showdifference.py runs one asexual and one sexual gene pool next to eachother. Compare.py runs both asexual and sexual algorithms on 50 gene pools for 2.5 minutes before creating a graph demonstrating the difference in performance. You can find my first result of Compare.py at ResultImageExample.png.
